Understanding Image Compression

In this first example, I uploaded my photo as it exported directly from Adobe Lightroom. This image is 1.3MB and represents the majority of information stored on my website. The image is larger than all the text on the website.

This second example was compresssed using tinyPNG. This image is only 10% of the original image's size and looks almost exactly the same when displayed on the web. This also means that the photo will load 10 times faster!

The final example has been run through a dithering engine to compress the image further. If you are going for a retro look, this can be a way to compress the image, and give it a throwback look.
